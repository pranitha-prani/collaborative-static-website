# Batch 3 Portfolio Website

This is a **simple website** created by **Batch 3** to showcase our skills, projects, and learning journey in Web Development and DevOps.

## Project Overview
The website contains multiple pages, each handled by a different team member. The goal is to create a professional, simple website.

## Pages and Team Responsibilities

- **Home** – Pranitha  
  Includes: Navigation header, profile section, and an intro video.

- **About Us** – Vimala  
  Includes: Team images and short descriptive.

- **Skills** – Surendar  
  Includes: Skills overview, certificates, internships, and other achievements.

- **Contact Us** – Nivash  
  Includes: LinkedIn, GitHub, contact form, and email details.

- **What We Learned** – Felix  
  Includes: Topics learned such as Linux, Git & GitHub, HTML, CSS, and practical exercises.

- **Testimonial** – Nivya  
  Includes: Testimonials for all team members.

## Styling
All pages use a **common stylesheet** (`style.css`) for consistent look and feel.

## Git Workflow
- Each page has its **own branch** in the repository.  
- Team members are responsible for developing their pages independently.  
- Once all pages are complete, branches will be merged for the final demo in main branch.
